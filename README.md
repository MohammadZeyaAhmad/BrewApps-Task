## APIs:

2.Get Book by Id:Method:GET  
 endpoint:/api/v1/BrewApps/book/:id
3.Add Book:Method:POST
  endpoint:/api/v1/BrewApps/book
4.Update Book:Method:PATCH
  endpoint:/api/v1/BrewApps/book/:id
5.Delete Book:Method:DELETE
  endpoint:/api/v1/BrewApps/book/:id
